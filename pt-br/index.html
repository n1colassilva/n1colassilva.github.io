Site de APIs

Site que utiliza várias APIs que mostram várias informações de utilidade questionável

Todas não utilizam chaves privadas para preservar a simplicidade do projeto.

  • A primeira mostra piadas, maioria com o tema de programação.
  • A segunda mostra fatos realmente inúteis. Pelo menos são interessantes!
  • A terceira infelizmente não permite muitas chamadas, a última mostra conselhos aleatórios.

Este projeto foi muito útil para aprender sobre o básico de como utilizar APIs, sem querer fazer isso parecer um trabalho escolar mas foi interessante ver como elas podem ser utilizadas para enviar e reber informações.

CitiesBot

CitiesBot é um projeto que criei para automatizar a minha jogatina de rotina no game "Cities War".

É um script em Python que utiliza Selenium para realizar simples tarefas que envolvem simplesmente clicar em elementos da página e esperar ela carregar.

Ignorando que foi meu primeiro projeto na língua o código está caótico por detrás da pseudo-interface que criei em um outro arquivo, inicialmente a mira era ser uma ferramenta que utiliza a linha de comando do python, mas com minhas abilidades da época essa ideia estava violando a regra de manter tudo simples e funcional.

Devido a natureza das atualizações do site o script para de funcionar após cada atualização, ele é bem sensível a falhas de configuração do usuário, talvez algum dia eu tento criar uma interface.

Levando o problema acima em conta, acredito que deixei simples manter o código, ele está cheio de comentários explicando o que faz e o porque de cada parte.
