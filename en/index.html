CitiesBot

CitiesBot is a script i created for automating the mundane tasks of a game i enjoy called "Cities War".

It's a script written in python that uses selenium to automate clicking, waiting for an element to pop up and calculating what to sell and how much

As my first python project it is quite rough, it was was planned to be a command line tool, but making that sounded too complicated, so i decided on making it just a script that requires a configuration file to be set up.

Sadly due to the nature of the game's updates the script is both hard an easy to update, hard because every. single. update. requires a remapping of where the elements are located, something will usually break no matter what.

Though in my great wisdom i saw that happening and tried to keep everything simple to update and well documented so any maintainer can just locate and update the positions.
